### Build
```bash

```

### Generate Code
```bash

```

## Why "BeaverSpec"?

Beavers are nature's engineers, known for building robust, well-structured dams. Like a beaver, this tool helps you build solid, reliable API implementations from specifications. The CLI name "beaver" is short, memorable, and easy to type.
